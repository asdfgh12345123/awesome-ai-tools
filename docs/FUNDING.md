# Funding

This project is free to use.

Sponsorship is optional and helps maintain the directory, check links, update resources, and improve documentation.

## Supported platforms

- **Ko-fi** is recommended for international users: https://ko-fi.com/aitoolshub
- **爱发电** is recommended for users in China: https://ifdian.net/a/aitoolshub

## Important notes

- Sponsorship does not unlock paid-only features.
- Sponsorship does not change public access to the project.
- This project does not promise rankings, traffic, income, or personal services in return.

## Supporter display

Public supporters may be displayed by display name or initials only. If a supporter prefers not to be shown, they can mention it in their support message or ask to be removed later.

## Privacy

No supporter information is sold. Supporter names are used only for acknowledgement if they choose to be listed.
