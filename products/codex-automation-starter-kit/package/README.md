# Codex Automation Starter Kit - Lite Version

This is the **free preview** of the Codex Automation Starter Kit.

## What is this?

A practical starter kit for organizing local projects into automated workflows using OpenAI Codex and GitHub Actions.

## What you get in this Lite version

| File | Description |
|------|-------------|
| README.md | This file |
| checklist.md | Step-by-step project organization checklist |
| codex-prompts.md | Ready-to-use prompts for Codex CLI |
| github-actions-template.md | GitHub Actions YAML templates |
| windows-commands.md | PowerShell and Git command reference |
| token-safety-checklist.md | Security best practices for API tokens |
| examples/example-project-audit-report.md | Example audit report from Codex |
| examples/example-codex-handoff-prompt.md | Example handoff prompt for Codex sessions |
| examples/example-github-actions-build.yml | Example GitHub Actions workflow file |

## What is NOT in this Lite version

The **Pro version** (available via Ko-fi or Afdian) includes additional content:

- Complete working Python scripts (build_site.py, generate_sitemap.py, check_links.py, generate_stats.py)
- More real-world examples and case studies
- Advanced Codex prompts for complex workflows
- Troubleshooting guide for common errors
- Ready-to-customize HTML templates

## Who is this for?

- Developers using Windows who want to set up Codex CLI for local project automation
- Self-learners who want a structured approach to AI-assisted project maintenance
- Anyone who wants to organize their GitHub repositories with proper CI/CD

## Who is this NOT for?

- People looking for passive income or guaranteed earnings
- Users who expect zero-effort automation
- Anyone unfamiliar with basic Git and command line usage

## Getting started

1. Read this README
2. Follow the checklist in `checklist.md`
3. Try the prompts in `codex-prompts.md`
4. Adapt the GitHub Actions template to your project
5. Review the examples for real-world usage patterns

## Important notes

- This Lite version is free and open source
- The Pro version is a paid product delivered via Ko-fi or Afdian
- This kit does not guarantee income or passive earnings
