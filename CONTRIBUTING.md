# Contributing to Awesome AI Tools

Thank you for helping improve this project.

## Ways to contribute

### Submit an AI tool
1. Open a **Tool submission** issue.
2. Include the tool name, URL, category, and a short description.
3. Use pricing labels such as Free, Freemium, Paid, or Unknown.

### Submit prompts
1. Open a **Prompt submission** issue.
2. Include prompt title, category, example prompt text, and a short use case.

### Submit RAG resources
1. Open a **Resource submission** issue.
2. Include the project name, URL, and why it is useful for retrieval-augmented generation.

### Submit AI agent frameworks
1. Open a **Resource submission** issue.
2. Include the project name, URL, category, and a short description.

### Report a broken link
1. Open a **Broken link** issue.
2. Include the page URL, the broken link, and what you expected to see.

## Data format tips

If you are submitting structured data directly, include:
- `name`
- `url`
- `category`
- `pricing`
- `description`
- `tags`
- `open_source`
- `last_checked`

## Pull request checklist

- Keep changes focused.
- Do not remove existing working URLs.
- Avoid exaggerated marketing claims.
- Verify new links open correctly.
- If you update generated content, run the local validation scripts when possible.
