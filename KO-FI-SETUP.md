# Ko-fi Setup Guide

## Step 1: Complete Profile

Go to https://ko-fi.com/aitoolshub/settings

- Display Name: AI Tools Hub
- Profile Picture: Upload AI-themed icon
- Cover Image: 1200x400px banner
- About:

```
I maintain a free AI Tools Hub with 300+ curated resources.

What I offer:
- AI Tools Directory (200+ tools, updated weekly)
- ChatGPT Prompt Library (500+ tested prompts)
- AI Model Comparison
- Awesome Lists for AI Agents, RAG, and more

GitHub: https://github.com/asdfgh12345123
Website: https://asdfgh12345123.github.io/awesome-ai-tools/
```

## Step 2: Set Up Payment

Go to https://ko-fi.com/aitoolshub/settings/payment
Connect PayPal account

## Step 3: Create Tiers

| Tier | Price | Perks |
|------|-------|-------|
| Coffee | $3 | Name on supporters page |
| Supporter | $5/mo | Name in README, early access |
| Hero | $15/mo | Tool requests, priority support |
| Champion | $50/mo | Logo on homepage, monthly call |
